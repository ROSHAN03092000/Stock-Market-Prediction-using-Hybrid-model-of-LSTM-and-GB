Requirements:
install anaconda navigator, you can run code either in jupyter notebook or spyder
install sklearn libraries, Keras libraries, numpy, pandas, seaborn and matplotlib libraries
installation process:
	pip install keras
	pip install scikit-learn
	pip install numpy
	pip install pandas
	pip install seaborn
	pip install matplotlib
Once the installation process of the library is finished, make sure that csv file that you're importing are located properly
Run the program.....
